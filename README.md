# unity-cypress

unity-cypress is a repo for Cypress part of homework 2.

## Setup
1. Clone repo 
`git clone https://gitlab.cs.ttu.ee/htoots/unity-cypress`
2. Install necessary modules with `npm i`
3. Open cypress to run tests with `npm test`
